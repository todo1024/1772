课程介绍：

随着元宇宙、数字藏品以及区块链应用的迅速发展，科技界和投资界燃起了对Web3.0的巨大热情。Web 3.0巨头孵化器ConsenSys在四个月内估值翻倍。全球科技巨头正在加速布局互联网的下一个阶段：Web3.0。
